## Getting Started
### Clone the repository:​
```bash
git clone https://github.com/CableKungZ/coe-backendPrograming.git
```
### Navigate to the project directory:​
```bash
cd coe-backendPrograming
```
### Install dependencies:​
```bash
npm install
```
### Run the application:​
```bash
npm start
```
